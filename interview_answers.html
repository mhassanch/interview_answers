What is a RESTful API and how would you document it?

The primary goal of API is to standardize data exchange between web services. Depending on the type of API, the choice of protocol changes. On the other hand, REST API is an architectural style for building web services that interact via an HTTP protocol.
While documenting RESTful API, one should describe the following points:
1. various endpoints available
2. their methods
3. parameters
4. and other details

And he should also document sample responses from the endpoints.

Can you write simple server in Node.js that returns Hello World?

Yes, first i installed node in my system and then in the folder where i wanted to create server i created a file named app.js and then i wrote the code in the file. and then i run the server by typing node app.js in the terminal. The code is given below.

const http = require('http');

const hostname = '127.0.0.1';
const port = 3000;

const server = http.createServer((req, res) => {
res.statusCode = 200;
res.setHeader('Content-Type', 'text/html');
res.end('index.html');
});

server.listen(port, hostname, () => {
console.log(`Server running at http://${hostname}:${port}/`);
});
